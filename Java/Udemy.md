# Java Udemy Notes

### Priminitive Types

* boolean
* byte (**8 bits**)
    * Byte.MIN_VALUE **-128** / Byte.MAX_VALUE **127**
* char
* short (**16 bits**)
    * Short.MIN_VALUE **-32768** / Short.MAX_VALUE **32767**
* int (**32 bits**)
    * Integer.MIN_VALUE **-2147483648** / Integer.MAX_VALUE **2147483647**
    * Underflow / Overflow
* long (**64 bits**)
    * 100L, force Java to treat the value as long
* float
* double
